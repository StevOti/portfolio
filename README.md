# Stephen Otieno — Portfolio

This repository contains a static portfolio website for Stephen Otieno. The site showcases projects, services, and contact details using plain HTML, CSS, and vanilla JavaScript.

## Features

- **Responsive layout:** Desktop grid and a mobile carousel for the portfolio section.
- **Pages included:** Portfolio, Services, About, Contact.
- **Project cards:** Clickable project cards that link to GitHub repositories.
- **Visuals & UX:** Hero sections, roll-underlines on nav links, sticky header, reveal animations, and a testimonial carousel.

## Preview locally

Open `index.html` directly in a browser, or run a simple HTTP server (recommended):

Windows (Command Prompt):

```
python -m http.server 8000
```

Windows (PowerShell):

```
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

If `python` is not on your PATH, try `py -m http.server 8000`.
