## Preview locally

Open `index.html` directly in a browser, or run a simple HTTP server (recommended):

Windows (Command Prompt):

```
python -m http.server 8000
```

Windows (PowerShell):

```
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

If `python` is not on your PATH, try `py -m http.server 8000`.
